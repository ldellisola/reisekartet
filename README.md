# Reiserkartet
Reisekartet (The travel map) is an open source travel destination manager built with Vue 3 and .NET 8.

## Installation

```bash
git clone https://github.com/ldellisola/Reiserkartet.git
cd Reiserkartet/docker
docker compose up
```
